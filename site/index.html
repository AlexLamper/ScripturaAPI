API Endpoints

Hieronder vind je een overzicht van alle API endpoints. Klik op een voorbeeld-URL om deze te openen, of kopieer de URL met één klik.

Endpoint Beschrijving Voorbeeld
/random Willekeurig vers /random
/verse Specifiek vers /verse?book=Genesis&chapter=1&verse=1
/passage Meerdere verzen uit één hoofdstuk /passage?book=Lukas&chapter=2&start=1&end=10
/books Lijst van boeken /books
/chapters Hoofdstukken in een boek /chapters?book=Lukas
/verses Verzen in een hoofdstuk /verses?book=Lukas&chapter=2
/search Zoek op tekst /search?query=vrede
/daytext Dagtekst (optioneel: ?seed=...) /daytext
/versions Beschikbare vertalingen /versions
/chapter Alle verzen van een hoofdstuk /chapter?book=Genesis&chapter=1
