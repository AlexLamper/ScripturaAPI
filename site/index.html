Hieronder vind je een overzicht van alle API endpoints. Klik op een voorbeeld-URL om deze te openen, of kopieer de URL met één klik.

Bekijk de API-documentatie (Swagger UI)
Endpoint Beschrijving Voorbeeld
/api/random Willekeurig vers
/api/random
/api/verse Specifiek vers
/api/verse?book=Genesis&chapter=1&verse=1
/api/passage Meerdere verzen uit één hoofdstuk
/api/passage?book=Lukas&chapter=2&start=1&end=10
/api/books Lijst van boeken
/api/books
/api/chapters Hoofdstukken in een boek
/api/chapters?book=Lukas
/api/verses Verzen in een hoofdstuk
/api/verses?book=Lukas&chapter=2
/api/search Zoek op tekst
/api/search?query=vrede
/api/daytext Dagtekst (optioneel: ?seed=...)
/api/daytext
/api/versions Beschikbare vertalingen
/api/versions
/api/chapter Alle verzen van een hoofdstuk
/api/chapter?book=Genesis&chapter=1
